# DigiSafe – Cybersecurity Awareness Website

**Project Type:** Static Website  
**Technologies:** HTML, CSS  
**Hosted on:** Vercel

---

## Overview

DigiSafe is a cybersecurity awareness website designed for university students. Its purpose is to educate students on protecting personal information online, understanding cyber threats, and adopting safe digital habits.

This project was created as part of the **CSC125** course and demonstrates practical skills in HTML and CSS.

---

## Features

- **Landing Page:** Introduces cybersecurity concepts with clear hierarchy and an embedded educational video.  
- **Navigation Bar:** Links to Home, About, Services, and Contact pages; includes a logo for branding.  
- **Cybersecurity Tips Section:** Organized grid layout covering passwords, phishing, and unsafe links.  
- **Newsletter Subscription:** Simple subscription form to encourage engagement.  
- **About Page:** Project mission and developer introduction.  
- **Contact Page:** Structured form with name, email, country, and message fields.  
- **Services Page:** Highlights information, advice, tools, and support offerings.  
- **Responsive Design:** Layouts optimized for multiple screen sizes.

---

## Learning Outcomes

- Built a multi-page static website using semantic HTML and centralized CSS.  
- Applied CSS Grid and Flexbox concepts for layout and responsiveness.  
- Implemented forms and embedded media for improved user experience.  
- Deployed the site on a free static hosting platform (**Vercel**).  
- Practiced documenting code and preparing a presentable portfolio project.
